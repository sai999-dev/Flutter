# Flutter Backend Package

Backend services and business logic for the mobile app.

## Structure

```
lib/
├── services/          # API services
├── storage/          # Storage services
├── utils/            # Utilities
└── models/           # Data models
```

## Usage

This package can be used by the frontend app or deployed as a separate service.

## Dependencies

- `http` - API communication
- `shared_preferences` - Local storage
- `flutter_secure_storage` - Secure token storage
